How it works — 3 steps

      
    
        
  1. Load your roster — Open an .xlsx/.csv, paste rows, or start from the Template. Edit on the Roster tab.
    2. 
        
  2. Build groups — go to the Groups tab, pick the unit (and class year), then Solve. Drag to adjust and lock students, then re-solve if needed.
    3. 
        
  3. Export — download the workbook. That file becomes your master — open it next unit so history carries forward.
    4. 
      

      
Groups are built automatically from the tutors you assign to a unit. The Schedule tab is optional.
